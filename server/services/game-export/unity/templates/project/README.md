1. **Requirements**: Unity 2022.3 LTS or newer (with New Input System package)

## Project Structure

```
Assets/
├── Scripts/
│   ├── Core/            Game manager, data loader
│   ├── Characters/      Player + NPC controllers
│   ├── Systems/         Action, combat, quest, inventory, crafting, etc.
│   ├── World/           Procedural generators
│   └── UI/              HUD, minimap, inventory, dialogue, quest tracker
├── Resources/
│   └── Data/            JSON data files (WorldIR, characters, actions, etc.)
├── Prefabs/             Create prefabs for Player, NPC, Buildings, etc.
├── Scenes/              Main world scene
└── Materials/           Create materials in the editor
```

## Data Loading

The game loads `Resources/Data/WorldIR.json` at runtime via
`InsimulDataLoader.LoadWorldData()`. Individual JSON files in
`Resources/Data/` can also be loaded independently.

## Notes

- Prefabs and Scenes must be created in the Unity Editor
- C# scripts provide all system logic; extend via inheritance or composition
- NPC AI uses Unity NavMesh — bake navigation after placing terrain
- UI uses Unity's Canvas system with TextMeshPro
- Input uses the New Input System (InputActions asset should be created in editor)
