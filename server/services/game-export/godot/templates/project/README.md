## Architecture

- **Autoloads**: GameManager, DataLoader, and all game systems are singletons
- **Scene tree**: Main scene has WorldEnvironment, DirectionalLight3D, Player, and generator nodes
- **Data loading**: `DataLoader.load_world_data()` reads `res://data/world_ir.json`
- **NPC AI**: Uses NavigationServer3D — bake NavMesh after placing terrain
- **Input**: WASD + Space (jump) + E (interact) + LMB (attack) + Esc (menu)

## Notes

- Models/textures must be imported via the Godot editor
- Extend scripts via inheritance or composition
- UI uses Godot's Control nodes (CanvasLayer)
